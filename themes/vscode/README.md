# Earthshine (VSCode)

## Local Installations

MacOS / Linux

```bash
   git clone https://github.com/edsonjaramillo/earthshine-vscode.git ~/.vscode/extensions/earthshine
```

Windows

```bash
   git clone https://github.com/edsonjaramillo/earthshine-vscode.git %USERPROFILE%\.vscode\extensions\earthshine
```

## Activating theme

Run Visual Studio Code. Earthshine Theme will be available from `File -> Preferences -> Color Theme` dropdown menu.
